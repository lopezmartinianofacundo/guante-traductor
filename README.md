Prototipo de guante inteligente para el reconocimiento de Lengua de Señas Argentina (LSA) utilizando un ESP32-S2, sensores flex y un MPU6050.
Actualmente se encuentra en fase de pruebas mediante simulación y prototipado, utilizando potenciómetros en reemplazo de los sensores flex para validar el sistema antes de la implementación física.
El diseño del circuito y del PCB fueron hechos en KiCad.
